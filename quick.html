QUICK SORT

Quick Sort is a sorting algorithm based on the Divide and Conquer that picks an element as a pivot and partitions the given array around the picked pivot by placing the pivot in its correct position in the sorted array.

Algorithm


  1. Select an element from the array as the pivot.

  2. Rearrange the array around the pivot. After partitioning, all elements smaller than the pivot will be on its left, and all elements greater than the pivot will be on its right. The pivot is then in its correct position, and we obtain the index of the pivot.

  3. Recursively apply the same process to the two partitioned sub-arrays (left and right of the pivot).

  4. The recursion stops when there is only one element left in the sub-array, as a single element is already sorted.
