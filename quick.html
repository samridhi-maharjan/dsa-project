QUICK SORT

QuickSort is a sorting algorithm based on the Divide and Conquer that picks an element as a pivot and partitions the given array around the picked pivot by placing the pivot in its correct position in the sorted array.

Algorithm

  1. Start with the first element in the array.
  2. Compare the current element with the next element.
  3. If the current element is greater than the next element, swap them.
  4. Move to the next element and repeat steps 2-3 until the end of the array is reached
  5. After each pass through the array, the largest element in the unsorted portion of the array will be in its correct position.
  6. Repeat steps 1-5 for the remaining unsorted portion of the array until no more swaps are needed.
